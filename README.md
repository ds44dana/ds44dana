# Stella X

**Independent researcher building measurement infrastructure for AI reasoning quality.**

I study what happens when LLMs have to think in sequence — when diagnostic information arrives one clue at a time, the way it does in actual clinical practice. Standard accuracy benchmarks miss the failure mode I care about: models that find the right answer and then lose it.

---

### Current Work

**SIPS Measurement Framework** — Sequential Information Prioritization Scaffold
A structured prompting architecture and scoring system for auditing AI reasoning pathology in sequential clinical decision-making.

- 5+2 rubric: 7-dimensional scoring beyond binary accuracy
- 6-code failure taxonomy: KV, RF, SD, PC, LM, CR
- Validated across Claude Sonnet 4, GPT-4o, and Llama 3.3-70B (N=150 runs)
- Paper 1 in pre-submission state → arXiv [link coming]
